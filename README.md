# Mapper-example

Mapper DAO example. Yup another svbtle/obtvse clone cuz I'm simple like
that.

## Install

Install simple Bash build tool to run `Bakefile`

    npm install bake-bash -g
    npm install -d

See all tasks

    bake

Create database (ensure MySQL is running), put in root password

    bake create_db

Run server

    bake run
